# Contributing

Thanks for your interest. Here are the rules.

## What belongs here

- Tools you've actually used or can vouch for
- Free resources with real educational value
- Paid tools that are industry standard (mark them as paid)
- Links that work

## What doesn't belong

- Affiliate links
- Tools with no documentation
- Paywalled content with no free tier (unless it's an industry standard)
- Duplicates of existing entries
- AI-generated listicle dumps

## How to contribute

1. Fork the repo
2. Add your resource to the correct section
3. Follow the existing format: `[Tool Name](URL) — One-line description.`
4. Submit a PR with a brief note on why you're adding it

## Format

```
- [Tool Name](https://example.com/) — What it does in one sentence. Note if it's free, freemium, or paid.
```

Keep descriptions short and factual. No promotional language.
